# Notion 페이지 나눔 미리보기 (Notion PDF Preview)

Notion 페이지를 **내보내기(PDF) 전에** A4/A3 용지 기준으로 어느 줄이 몇 페이지에 들어가는지 눈으로 확인하는 크롬 확장(MV3)입니다. 미리보기 화면에서 바로 PDF로 저장하면 **보이는 그대로 출력**됩니다(WYSIWYG).

## 왜?

Notion의 기본 PDF 내보내기는 서버에서 렌더링되어, 내보내기 전엔 페이지 배치를 알 수 없습니다. PDF를 열어봐야 확인되니 "내보내기 → 열기 → 수정"을 반복하게 됩니다. 이 확장은 그 피드백 루프를 없애고, 편집 후 버튼 한 번으로 페이지 경계를 미리 봅니다.

## 설치 (개발자 모드)

1. `chrome://extensions` 접속 → 우측 상단 **개발자 모드** 켜기
2. **압축해제된 확장 프로그램을 로드** → 이 폴더 선택
3. 툴바에서 확장을 고정(핀)해두면 편리

## 사용

1. `app.notion.com`에서 대상 페이지 열기
2. 확장 툴바 버튼 클릭 → 새 탭에 **페이지 나눔 미리보기**가 열림
3. 상단에서 **용지(A4/A3)**·**여백** 조절 → 즉시 재계산
4. **PDF로 저장** → 그 미리보기 그대로 출력
   - 인쇄 대화상자에서 여백/머리글이 남으면 **여백: 없음, 머리글/바닥글: 해제**

## 동작 원리

- **자체 페이지 분할기**: 블록을 mm 단위 A4/A3 페이지 박스에 순서대로 흘려 담고, 그 페이지 div를 그대로 인쇄 → 미리보기 = 출력.
- **표/큰 블록 행 단위 분할 (v2)**: 한 페이지보다 큰 표는 **행(`<tr>`) 단위로 나눠** 여러 페이지에 이어 담고, 이어지는 페이지마다 헤더(`<thead>`)를 반복합니다. 표 외 컨테이너(리스트·컬럼 등)는 자식 요소 단위로 분할. 페이지보다 큰 분할 가능 블록은 **현재 페이지 남은 공간부터** 채웁니다.
- **표 폭 맞춤 (v2)**: Notion 표는 컬럼이 고정 px라 여백이 크면 오른쪽이 잘립니다. 표를 본문 폭에 맞게 축소(`width:100%` + 셀 `min-width:0` + 자동 줄바꿈)해 여백과 무관하게 항상 페이지 안에 들어오게 합니다(Notion export와 동일한 처리).
- **빈 레이어 스킵 (v2)**: Notion 편집 UI 잔재(높이 0 블록 핸들 오버레이, 표와 같은 높이의 내용 없는 배경/선택 오버레이 레이어 등)는 페이지를 차지하지 않게 건너뜁니다(빈 페이지·표 뒤 공백 방지).
- **이미지 인라인**: Notion 이미지는 인증 프록시→프리사인 S3로 리다이렉트되어 확장 페이지에서 직접 로드가 막힙니다. 백그라운드 서비스워커(host 권한으로 CORS 우회)가 fetch해 `data:` URL로 박제합니다.
- **스타일 재현**: Notion 자체 CSS를 스냅샷해 미리보기에 적용, 인쇄용 `@page`만 우리가 통제.

## 한계

- 아주 긴 **단일 텍스트 블록**(쪼갤 요소 자식이 없는 문단 등)이 한 페이지를 넘으면 잘림(`⚠` 표시).
- **DB(데이터베이스/컬렉션 뷰)**: 데이터 표는 나오지만 **편집용 UI(툴바·"새로 만들기"·"+"/"..." 버튼·"새 페이지" 행)가 함께 스냅샷**되어, 페이지보다 넓으면 오른쪽 버튼이 잘려 보입니다. Notion export처럼 이 편집 UI를 걷어내는 작업은 **미구현(예정)**. 문서·단순 표 위주 페이지에 최적.
- 표 한 행 자체가 한 페이지보다 크면 그 행은 잘림.
- 대상은 **Notion 웹(app.notion.com)**. 데스크톱(Electron) 앱은 범위 밖.
- 짧은 시간에 반복 실행하면 Notion 서버가 레이트리밋(429)할 수 있습니다(잠시 후 해제).

## 파일 구성

| 파일 | 역할 |
|---|---|
| `manifest.json` | MV3 매니페스트 (권한·액션) |
| `background.js` | 서비스워커: 버튼 클릭 시 스냅샷 주입, 이미지 fetch(CORS 우회) |
| `content-snapshot.js` | Notion 페이지에서 본문+CSS 스냅샷, 이미지 인라인 위임 |
| `preview.html` / `preview.js` | 미리보기 UI + 자체 페이지 분할기 + 인쇄 |
